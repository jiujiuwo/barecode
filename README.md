# barecode

## 先安装 Node.js,然后执行以下命令
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install
```

### 运行
```
cnpm run serve 
```

### 构建
```
cnpm run build
```

### 代码格式修复
```
cnpm run lint
```cle
